# AscensionPTBR

Projeto: Addon de tradução para Português Brasileiro (PT-BR)

Contexto

A partir de um addon de tradução que convertia entre DE/EN e FR, este repositório foi criado inspirando-se e tomando-o como referência para desenvolver um addon de tradução voltado a PT-BR. 

Objetivo

- Adaptar a arquitetura e funcionalidades do addon original para suportar traduções e localização para PT-BR.
- Implementar integração com serviços de tradução, ajustes de terminologia e melhorias de experiência específicas para o público brasileiro.
